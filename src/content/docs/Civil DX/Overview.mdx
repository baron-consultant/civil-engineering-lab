# Civil DX 플랫폼 개요

이 문서는 텍스트 위주의 설명을 보여줍니다.

## 주요 내용

Civil DX는 토목 엔지니어링의 디지털 전환을 이끄는 핵심 플랫폼입니다. BIM 데이터를 활용하여 설계부터 시공, 유지관리까지 전 과정의 효율성을 극대화합니다.

### 1. 데이터 통합
다양한 포맷의 엔지니어링 데이터를 하나로 통합하여 단일 모델에서 관리합니다. 이를 통해 부서 간 협업이 원활해지고 데이터 누락을 방지할 수 있습니다.

### 2. 자동화 도구
반복적인 설계 작업을 자동화하여 엔지니어의 피로도를 줄이고 창의적인 업무에 집중할 수 있도록 지원합니다.

### 3. 시각화
복잡한 3D 모델을 웹에서 바로 시각화하여 발주처나 시공사와의 커뮤니케이션을 돕습니다.
