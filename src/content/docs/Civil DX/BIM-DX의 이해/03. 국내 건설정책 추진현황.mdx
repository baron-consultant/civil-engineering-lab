## 1. 건설 정책 로드맵 및 목표

정부는 건설산업의 디지털 전환을 지속 추진하고 있으며, 현재는 **디지털화(BIM)와 자동화(OSC)** 를 핵심 목표로 삼고 있다.

![건설 정책 로드맵](/assets/images/04.png)

### 📅 주요 정책 추진 경과

| 시기 | 주요 추진 내용 |
| :--- | :--- |
| **~2007** | • 제3차 건설사업정보화 기본계획 (국토교통부)<br/>• 가상건설시스템개발연구단 (KAIA) |
| **2008~2010** | • 제4차 건설사업정보화 기본계획<br/>• 건설공사 시공상세도 작성지침 |
| **2012~2015** | • 제5차 건설기술진흥기본계획<br/>• BIM/GIS 플랫폼기반 건설공간정보 통합운영 기술개발 |
| **2016~2017** | • 도로분야 BIM 시범 적용 (설악~청평)<br/>• 제6차 건설기술진흥기본계획<br/>• 건설공사 표준시방서 체계 정리 |
| **2018** | • 스마트 건설기술 로드맵 수립<br/>• 고속도로공사 전문시방서 (한국도로공사)<br/>• 건설산업 혁신방안 |
| **2020** | • **건설산업 BIM 기본지침** (국토교통부)<br/>• 전면 BIM 설계 발주<br/>• 스마트건설기술개발사업 착수 |
| **2021** | • **스마트건설기술 활성화 지침**<br/>• BIM기반 건설산업 디지털 전환 로드맵<br/>• XR 플래그십 프로젝트 (건설분야) |
| **2022** | • **스마트건설 활성화 방안** (2022.7)<br/>• 건설산업 BIM 시행지침 및 적용지침 |
| **2023~** | • 제6차 건설공사지원 통합정보체계 기본계획(안)<br/>• 제7차 건설기술진흥기본계획<br/>• 고속도로 BIM 적용지침 (시공자편) |

## 2. 스마트 건설 활성화 방안

2022년 7월 국토교통부가 발표한 **'스마트 건설 활성화 방안'**은 건설산업의 디지털화와 자동화를 위한 핵심 전략을 포함한다.

![스마트 건설 활성화 방안](/assets/images/05.png)

### 핵심 키워드

| 구분 | 주요 내용 |
| :--- | :--- |
| **BIM**<br/>(Building Information Modeling) | **건설산업의 자동화, 지능화 등 스마트 건설을 실현하기 위한 기본 도구**<br/>건설정보 기반의 Process와 Products를 제공하여 디지털 전환의 기반을 마련함 |
| **OSC**<br/>(Off-Site Construction) | **탈현장 건설 (공장 제작 시공)**<br/>단위부재 또는 유닛(여러 부재가 합쳐진 모듈 등)을 공장에서 사전 제작한 이후, 현장으로 이동하여 레고 블럭처럼 조립하는 방식 |

## 3. 외국의 BIM 주요 현황

주요 선진국들은 정부 주도 혹은 민간 중심으로 BIM 도입을 적극적으로 추진하고 있다.


### 국가별 추진 특징

| 국가 | 추진 방식 | 주요 내용 |
| :--- | :--- | :--- |
| **영국** | **정부 주도**<br/>(공공 의무화) | • 2011년 "Government Construction Strategy"를 통해 **공공분야 BIM 의무화** 계획<br/>• ISO 19650 도입으로 글로벌 표준화 선도<br/>• 런던 Crossrail(광역급행철도) 프로젝트 등 대형 사업 적용 |
| **미국** | **민간 중심** | • 연방정부 차원의 의무화 대신 **민간 중심의 BIM 도입**<br/>• 정부와 AIA(미 건축가 협회) 등에서 가이드라인을 발표하여 공공 및 민간부문 지원 |
| **싱가포르** | **정부 주도**<br/>(강력한 지원) | • 정책수립과 기술지원을 통해 강제적이고 강력한 **정부 주도의 BIM** 추진<br/>• 공공인프라(지하철, 도로, 수자원) 프로젝트에 BIM 의무화 |

### 🌍 글로벌 트렌드 요약
> * 초기에는 건축분야 중심이었으나 현재는 **도로, 철도, 교량 등 공공인프라 분야로 확장**되는 추세임.
> * 공공 프로젝트에서 선도적으로 BIM을 도입한 후 **민간분야로 확산**되는 형태를 보임.
> * 단순 모델링을 넘어 **데이터의 디지털화**, 프로젝트의 **비용절감 및 효율성 확보**를 목표로 함.
