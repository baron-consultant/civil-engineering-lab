- 단방향 무한 선을 그립니다.
- 시작점을 기준으로 한쪽 방향으로 무한히 향하는 직선을 그립니다.

### 명령어 옵션

![][image1]

| 옵션       | 설명                                                                                                                                                                                                  |
| :--------- | :---------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------- |
| Bisect     | vertex point와 angle start point을 지정하여 angle end point과 angle start point간의 각도의 절반에 해당하는 각도(단방향 무한선과 angle start point간의 각도 \<= 90)를 가지는 단방향 무한선을 그립니다. |
| Horizontal | 마우스 커서에서부터 오른쪽으로 향하는 수평선을 그립니다.                                                                                                                                              |
| Vertical   | 마우스 커서에서부터 위쪽으로 향하는 수직선을 그립니다.                                                                                                                                                |
| Angle      | 입력한 각도를 가지는 단방향 무한선을 그립니다.                                                                                                                                                        |
| Offset     | 선택한 직선과 평행하고 입력한 Offset만큼의 거리를 가지는 단방향 무한선을 그립니다.                                                                                                                    |

#### Offset 키워드 입력 후 평행 단방향 무한선 그리기 키워드

![][image2]

| 옵션                       | 설명                                                                                                                                 |
| :------------------------- | :----------------------------------------------------------------------------------------------------------------------------------- |
| Through Point              | 직선 객체와 평행할 단방향 무한선의 위치를 마우스로 자유롭게 지정합니다.                                                              |
| Through                    | 화면 상에 마우스로 두 점을 지정하고 그 두 점 사이의 거리를 직선 객체와 평행할 단방향 무한선 간의 거리로 지정합니다.                  |
| 키워드 입력 없이 거리 입력 | 직선 객체와 평행할 단방향 무한선 간의 거리를 지정하고 이후 마우스로 직선 객체를 기준으로 단방향 무한선의 상하좌우 위치를 지정합니다. |

## **1\. 단방향 무한선 그리기 방법** [**시연 영상**](https://youtu.be/bbuZeackg44)

1\) “RAY” 명령어를 실행합니다.  
2\) 화면 상에 마우스로 무한선의 시작점을 지정합니다.  
3\) 화면 상에 마우스로 무한선의 Direction을 지정하면 단방향 무한선이 그려집니다.  
4\) ENTER 또는 ESC 키를 눌러 단방향 무한선 그리기를 종료합니다.

## **2\. 두 점을 이용한 단방향 무한선 그리기 방법** [**시연 영상**](https://youtu.be/GIc0rphN0ro)

1\) “RAY” 명령어를 실행합니다.  
2\) Bisect 옵션을 입력합니다.  
2\) 화면 상에 마우스로 무한선의 시작점을 지정합니다.  
3\) 화면 상에 마우스로 angle start point와 angle end point를 지정하면 단방향 무한선이 그려집니다.  
4\) ENTER 또는 ESC 키를 눌러 단방향 무한선 그리기를 종료합니다.

## **3\. 수직/수평 무한선 그리기 방법** [**시연 영상**](https://youtu.be/dNnLNcK45Xk)

1\) “RAY” 명령어를 실행합니다.  
2\) Horizontal 키워드(수평) 또는 Vertical 키워드(수직)를 입력합니다.  
3\) 마우스로 화면 상에 시작점을 지정하면 수평 또는 수직의 단방향 무한선이 그려집니다.  
4\) ENTER 또는 ESC 키를 눌러 단방향 무한선 그리기를 종료합니다.

## **4\. 특정 각도를 가진 단방향 무한선 그리기 방법** [**시연 영상**](https://youtu.be/mRljmkgV7CQ)

1\) “RAY” 명령어를 실행합니다.  
2\) Angle 키워드를 입력합니다.  
3\) 원하는 각도를 입력하고 ENTER를 누릅니다.  
3\) 마우스로 화면 상에 시작점을 지정하면 입력한 각도를 가진 단방향 무한선이 그려집니다.  
4\) ENTER 또는 ESC 키를 눌러 단방향 무한선 그리기를 종료합니다.

## **5\. 직선 객체와 평행한 단방향 무한선 그리기 방법** [**시연 영상**](https://youtu.be/oSnUug9NLdk)

1\) “RAY” 명령어를 실행합니다.  
2\) Offset 키워드를 입력합니다.  
3\) 평행 단방향 무한선을 그릴 옵션을 선택합니다.  
&nbsp;&nbsp;&nbsp;&nbsp;3-1) 각도를 입력하고 ENTER를 누릅니다.  
&nbsp;&nbsp;&nbsp;&nbsp;3-2) 마우스로 화면 상에 두 점을 지정하여 거리를 지정합니다.  
&nbsp;&nbsp;&nbsp;&nbsp;3-3) Through Point 키워드를 입력합니다.  
4\) 그려질 단방향 무한선과 평행할 직선 객체를 선택합니다.  
5\) 마우스로 단방향 무한선의 위치를 지정합니다.  
6\) ENTER 또는 ESC 키를 눌러 단방향 무한선 그리기를 종료합니다.
