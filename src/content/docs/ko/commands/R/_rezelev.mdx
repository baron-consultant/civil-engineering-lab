* 선택한 객체의 Z 값을 0(기본) 또는 지정한 값으로 일괄 리셋합니다.

![][image1]

## **1\. 객체 Z 값 변경하기**

1\) 명령창에 “REZELEV” 를 입력합니다.  
2\) Z 값을 리셋할 객체를 선택(선, 폴리선, 호, 원, 문자 등 2D 위주) 합니다.  
3\) 새 Z 값을 입력합니다. (기본: 0\)  
4\) 선택 객체의 Z가 입력값 으로 변경됩니다.  
![][image2]
