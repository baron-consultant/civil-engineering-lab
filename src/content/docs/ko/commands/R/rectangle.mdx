[image1]: ../../../../../assets/images/commands/rectangle/image1.png

- 폴리선으로 사각형을 생성합니다.
- 속성 값을 설정하여 사각형을 그릴 수 있습니다.

![][image1]

- RECTANGLE의 설정 옵션 키워드

* Chamfer : 사각형의 모서리를 지정된 거리만큼 대각선으로 자릅니다.
* Elevation : XY 평면으로부터 Z축 방향으로 그릴지 지정합니다.
* Fillet : 사각형의 모서리를 지정된 반지름 값만큼 둥글게 처리합니다.
* Rotated : 특정 각도로 회전시켜 생성합니다.
* Square : 정사각형을 그릴 때 사용합니다.
* Thickness : 사각형에 Z축 방향으로 두께를 부여하여 생성합니다.
* Width : 특정 폭으로 생성합니다.

## **1\. 사각형 그리기** [**시연 영상**](https://youtu.be/FbsJL06fZas)

1\) “RECTANGLE” 명령어를 실행합니다.
2\) 사각형의 한 모서리인 첫 번째 점을 마우스로 클릭합니다.
3)\ 대각선에 위치한 다른 모서리인 두 번째 점을 마우스로 클릭합니다.
