### Helix 설정 옵션 키워드

![][image4]

| 옵션          | 기능 설명                                                            |
| :------------ | :------------------------------------------------------------------- |
| Axis endpoint | 나선 스프링의 축이 끝나는 지점을 화면 상에 마우스로 직접 지정합니다. |
| Turns         | 나선의 회전 횟수를 설정합니다.                                       |
| Turn height   | 나선의 높이를 설정합니다.                                            |
| Twist         | 나선의 회전 방향을 설정합니다. (시계방향/시계반대방향)               |

\* Turns, Turn Height 두 옵션을 모두 설정한 경우 해당 옵션의 값에 따라 나선 스프링의 높이가  
자동으로 설정됩니다.

<br />

## **1\. 나선 스프링 그리는 방법** [**시연 영상**](https://youtu.be/MoHB-RHOkBc)

1\) “HELIX” 명령어를 실행합니다.  
2\) 화면 상에 마우스로 나선 스프링의 center point of base를 설정합니다.  
3\) 화면 상에 마우스로 지점을 클릭해 base radius를 설정하거나 명령어 창에 base radius를 입력합니다.  
4\) 화면 상에 마우스로 지점을 클릭해 top radius를 설정하거나 명령어 창에 top radius를 입력합니다.  
5\) 필요한 경우 나선 스프링의 옵션을 설정합니다.  
&nbsp;&nbsp;&nbsp;&nbsp;5-1) Axis endpont 키워드를 입력해 나선 스프링의 축이 끝나는 지점을 화면 상에 마우스로 지정합니다.  
&nbsp;&nbsp;&nbsp;&nbsp;5-2) Turns 키워드를 입력하고 명령어 창에 나선의 회전 횟수를 입력합니다.  
&nbsp;&nbsp;&nbsp;&nbsp;5-3) Turn Height 키워드를 입력하고 명령어 창에 나선의 높이를 입력합니다.  
&nbsp;&nbsp;&nbsp;&nbsp;5-4) Twist 키워드를 입력하고 나선의 회전 방향을 CW(시계방향) 또는 CCW(시계반대방향)로 설정합니다.  
6\) 화면 상에 마우스로 나선 스프링의 높이를 설정하거나 명령어 창에 높이를 입력 후  
ENTER를 누르면 명령어가 종료되고 나선 스프링이 그려집니다.
