- 타원을 폴리라인으로 그릴지 여부를 설정하는 명령어입니다.
- 설정 ON \=\> ELLIPSE 명령어로 그려지는 타원의 객체 타입은 폴리라인,  
  설정 OFF \=\> ELLIPSE 명령어로 그려지는 타원의 객체 타입은 타원입니다.

![][image1]

## **1\. 설정을 켜고 끄는 방법** [**시연 영상**](https://youtu.be/6wocHQQTr0g)

1\) “PELLIPSE” 명령어를 실행합니다.  
2\) ON 하고 싶은 경우 명령어 창에 ON 또는 1을 입력하고,  
OFF 하고 싶은 경우 명령어 창에 OFF 또는 0을 입력합니다.
