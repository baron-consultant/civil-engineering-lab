- 선분 위에 지정거리마다 수직선을 그립니다.
- 명령어를 실행하면 명령어 옵션 팝업창이 열립니다.
- 명령어 옵션 팝업창에서 “확인” 버튼을 누르면 옵션이 저장되고 수직선 넣기 동작이 시작됩니다.
- “초기화” 버튼을 누르면 명령어 옵션이 기본 옵션으로 초기화됩니다.

![][image1]

## **1\. 선분 객체에 지정 거리마다 수직선을 넣는 방법** [**시연 영상**](https://youtu.be/DrS2ZzvmHI4)

1\) “MPERPENDICULAR” 명령어를 실행합니다.  
2\) 명령어 옵션 팝업창에서 설정을 수정하고 “확인” 버튼을 누릅니다.  
3\) 화면 상에서 마우스로 수직선을 넣을 선분 객체, 기준점, 진행방향을 선택합니다.  
4\) 명령어 창에 이전 점으로부터 선 거리를 입력하고 ENTER를 누르면 수직선이 생성됩니다.  
5\) 수직선 넣기 작업을 종료하고 싶은 경우 ESC 키를 누르면 명령어가 종료됩니다.
