## **1\. 선스타일의 Scale 값 변경하기** [**시연 영상**](https://youtu.be/MBviatvK29I)

1\) 명령어 창에 CHLTYPE 을 입력 후, 직선(LINE) 객체를 선택합니다.

![][image1]

2\) HIDDEN 선이 시작할 지점을 클릭하거나 거리를 입력합니다.  
3\) HIDDEN 선이 끝날 지점(두번째 점)을 클릭하거나 거리를 입력합니다.

![][image2]

4\) 선스타일의 Scale 값을 입력합니다.
