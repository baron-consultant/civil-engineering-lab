- 폴리선으로 도넛을 생성합니다.
- 속성 값을 설정하여 DONUT을 그릴 수 있습니다.

### DONUT의 설정 옵션 키워드

![][image1]

| 옵션           | 기능 설명                            |
| :------------- | :----------------------------------- |
| 2Point         | 2점 지정으로 DONUT을 생성합니다.     |
| 3Point         | 3점 지정으로 DONUT을 생성합니다.     |
| RadTanTan(RTT) | 두 객체에 접하는 DONUT을 생성합니다. |

<br />

## **1\. 도넛 그리기** [**시연 영상**](https://youtu.be/saj9LwPB9zE)

1\) “DONUT” 명령어를 실행합니다.  
2\) Inside diameter를 입력합니다.  
3\) Outside diameter를 입력합니다.  
4\) 화면 상에 마우스로 클릭합니다.  
5\) ENTER or ESC 키를 눌러 그리기를 종료합니다.

## **2\. 도넛 그리기 \- 2Point** [**시연 영상**](https://youtu.be/Qhzvj2-QApU)

1\) “DONUT” 명령어를 실행합니다.  
2\) 2Point 키워드를 입력하고 Width를 입력합니다.  
3\) 첫번째 점을 마우스로 지정합니다.  
4\) 두번째 점을 마우스로 지정합니다.

## **3\. 도넛 그리기 \- 3Point** [**시연 영상**](https://youtu.be/1oAuFECMMsQ)

1\) “DONUT” 명령어를 실행합니다.  
2\) 3Point 키워드를 입력하고 Width를 입력합니다.  
3\) 첫번째 점을 마우스로 지정합니다.  
4\) 두번째 점을 마우스로 지정합니다.  
5\) 세번째 점을 마우스로 지정합니다.

## **4\. 도넛 그리기 \- RadTanTan(RTT)** [**시연 영상**](https://youtu.be/8cxAGDF_ncA)

1\) “DONUT” 명령어를 실행합니다.  
2\) RadTanTan 키워드를 입력하고 Width를 입력합니다.  
3\) 지름 diameter를 입력합니다.  
4\) 첫번째 객체 tangent point를 선택합니다. (객체는 Line,Circle,Arc, Polyline)  
5\) 두번째 객체 tangent point를 선택합니다. (객체는 Line,Circle,Arc, Polyline)
