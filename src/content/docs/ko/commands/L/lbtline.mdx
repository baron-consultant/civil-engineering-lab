- 두 선 사이에 등분선을 넣습니다.

- 명령어를 실행하고 설정(S) 키워드를 입력하면 명령어 옵션 팝업창이 열립니다. 이 팝업창에서 명령어의 동작 옵션을 설정할 수 있습니다

- “초기화” 버튼을 누르면 명령어 옵션이 기본 옵션으로 초기화됩니다.

- 명령 실행 중 등분 개수 입력이 체크되어 있으면 두 선을 등분할 개수를 명령어 실행시마다 입력하여 설정할 수 있고, 체크되어 있지 않으면 등분 개수 입력 설정에 따라 등분선 넣기가 실행됩니다.

- 콤보 박스로 등분선을 넣을 레이어를 지정하거나, ![image alt emoji][image2] 버튼을 눌러 화면 상에서 객체를 선택하고 그 객체의 레이어를 등분선을 넣을 레이어로 설정할 수도 있습니다.

## **1\. 두 선 사이에 등분선을 넣는 방법** [**시연 영상**](https://youtu.be/eKIeHFzzWTw)

1\) “LBTLINE” 명령어를 실행합니다.  
2\) 화면 상에서 등분선을 넣을 두 선을 선택합니다.  
3\) ENTER를 누르면 선택한 두 선 사이에 등분선이 생성되고 명령어가 종료됩니다.

## **2\. 등분선 넣기 설정을 수정하는 방법** [**시연 영상**](https://youtu.be/vQPHhnIlEp4)

1\) “LBTLINE” 명령어를 실행합니다.  
2\) 설정(S) 키워드를 선택/입력합니다.  
3\) 명령어 옵션 팝업창에서 원하는 설정으로 변경합니다.  
4\) “확인” 버튼을 누르면 설정이 저장됩니다.
